# 🏛️ Matdata Mitra — मतदाता मित्र

> *Your nonpartisan companion for understanding Indian democracy.*

Matdata Mitra ("Voter's Friend") is an interactive, AI-powered civic education 
web app that helps Indian citizens — especially first-time voters — understand 
the election process, timelines, key officials, political parties, and politicians 
in a simple and engaging way.

## ✨ Features

- 💬 **AI Chat Assistant** — Conversational guide powered by Claude AI, answering
  questions about elections, voting rights, and the Constitution
- 🗓️ **Election Timeline Explorer** — Interactive phase-by-phase breakdown of
  India's general election cycle
- 🏛️ **Officials Directory** — Profiles of the President, PM, CEC, and key
  constitutional offices
- 🎯 **Party Explorer** — Searchable cards for 20+ national and regional parties
  with ideology tags and leadership info
- 👤 **Politician Profiles** — Quick-search bios for top Indian politicians
- 📋 **Voter Checklist** — Step-by-step guide to casting your first vote
- ❓ **Civic Quiz** — AI-generated multiple choice quiz to test your knowledge
- 🌐 **Bilingual** — Full English / हिन्दी language toggle
- 🌙 **Dark mode** support

## 🛠️ Tech Stack

- **Frontend:** React (single-page, component-based)
- **AI:** Anthropic Claude API (`claude-sonnet-4-20250514`)
- **Icons:** lucide-react
- **Fonts:** Noto Sans (Devanagari + Latin)
- **Styling:** Inline CSS / CSS-in-JS — no external frameworks

## 🎨 Design

Minimalist and modern, inspired by India's tricolor identity — saffron, white,
and green with navy blue accents. Mobile-first and fully responsive.

## ⚠️ Disclaimer

Matdata Mitra is a nonpartisan civic education tool. It does not represent the
Election Commission of India or any political party. All information is provided
for educational purposes only.
